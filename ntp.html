NTP
(Network Time Protocol)

Install & Konfigurasi NTP

[root@localhost ~]# yum install ntp -y
[root@localhost ~]# nano /etc/ntp.conf
Cari baris server 0.centos.pool.ntp.org iburst , lalu tambahkan pagar didepannya
server 0, 1, 2, 3 dan dibawahnya tambahkan
server 3.id.pool.ntp.or
server 0.asia.pool.ntp.or
server 2.asia.pool.ntp.or

Cari baris # Hosts on local network..... , beri tanda pagar pada restrict pertama dan tambahkan script restict lagi
restrict ip network centos dan windows (ex. 172.16.11.0) netmask 255.255.255.0 nomodify no trap
Keluar dan Simpan

[root@localhost ~]# timedatectl set-timezone Asia/Jakarta

Membuka Koneksi (Firewall)

[root@localhost ~]# firewall-cmd --zone=public --permanent--add-service=ntp
[root@localhost ~]# firewall-cmd --reload

Menjalankan NTP

[root@localhost ~]# systemctl start ntpd
[root@localhost ~]# systemctl enable ntpd
[root@localhost ~]# systemctl status ntpd
Pastikan active(running)

Menyeting Windows

Buka Control Panel -> Clock, Language, and Region -> Date and Time
Buka Tab Internet Time -> Change settings...
Centang Synchronize... dan Isikan IP centOS -> Update Now -> OK
Pastikan IP Windows dan centOS sudah satu network dan bisa saling terkoneksi
